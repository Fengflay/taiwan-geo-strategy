## 一、市場背景與機會

### 台灣用戶 AI 使用現況

| AI 工具 | 台灣市佔估計 | 特性 |
|---------|------------|------|
| ChatGPT (OpenAI) | ~45% | 台灣最主流，付費用戶比例高 |
| Google Gemini | ~25% | 整合 Google 搜尋，企業用戶多 |
| Perplexity | ~10% | 成長快速，重度搜尋替代用戶 |
| Claude (Anthropic) | ~8% | 技術圈與企業用戶偏好 |
| Microsoft Copilot | ~7% | Office 365 企業用戶 |
| 其他（本地/開源） | ~5% | Line AI、台灣本土工具 |

### 核心機會

- **市場教育空白**：台灣 90% 以上企業不知道 GEO 是什麼，先行者紅利巨大
- **SEO 預算轉移**：企業已有 SEO 預算，GEO 是自然的預算轉移對象
- **AI 搜尋滲透加速**：台灣 25-45 歲用戶中，超過 60% 已將 AI 作為資訊獲取的主要管道之一
- **繁體中文內容稀缺**：相比簡體中文，繁體中文的高品質結構化內容少，更容易被 AI 引用

## 二、台灣 vs 中國大陸 GEO 差異

| 維度 | 中國大陸 | 台灣 |
|------|---------|------|
| **主流 AI 平台** | 豆包、DeepSeek、Kimi、文心一言 | ChatGPT、Gemini、Perplexity、Claude |
| **搜尋引擎** | 百度 | Google（市佔 >90%） |
| **語言** | 簡體中文 | 繁體中文（zh-TW） |
| **百科平台** | 百度百科 | 維基百科繁體中文版 |
| **問答社區** | 知乎 | PTT、Dcard、Mobile01 |
| **專業媒體** | 36氪、虎嗅 | 數位時代、科技新報、iThome、天下雜誌 |
| **社群平台** | 微信、微博、小紅書 | Facebook、Instagram、LINE、Threads |
| **B2B 社群** | 脈脈 | LinkedIn |
| **影音平台** | B站、抖音 | YouTube、Podcast |
| **法規環境** | 廣告法、反不正當競爭法 | 公平交易法、消費者保護法 |
| **內容風格** | 偏數據驅動、功能導向 | 重視信任感、案例故事、第三方背書 |

### 關鍵差異要點

1. **AI 平台完全不同**：台灣用戶使用的全是國際 AI 平台，這些平台更重視英文和繁中的高品質來源
2. **Google 生態系為核心**：Google 搜尋 + Gemini 整合是台灣的核心戰場
3. **信任建立方式不同**：台灣市場更重視第三方媒體背書、真實案例、國際認證
4. **繁體中文語義**：用詞習慣不同（如「軟體」vs「軟件」、「資料」vs「數據」），必須用台灣用語

- [ ] FAQ 頁面使用 FAQPage Schema

### 5.5 GEO 監測腳本概念

```python
# geo_monitor.py — 每日 AI 能見度監測
# 需搭配 OpenAI API / Google AI API

import openai
import csv
from datetime import datetime

BRAND_NAME = "你的品牌名"
QUERIES = [
    f"{BRAND_NAME} 是什麼",
    f"{BRAND_NAME} 評價",
    "你的行業 + 推薦工具",
    "你的行業 + 解決方案比較",
    "如何選擇 你的產品類型",
    # ... 擴展至 20-30 個核心問題
]

def check_ai_visibility(query):
    """檢查 AI 回答中是否提及品牌"""
    response = openai.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": query}]
    )
    answer = response.choices[0].message.content
    mentioned = BRAND_NAME.lower() in answer.lower()
    # 分析引用位置：前段(1-3句)、中段、後段
    sentences = answer.split('。')
    position = "未提及"
    if mentioned:
        for i, s in enumerate(sentences):
            if BRAND_NAME.lower() in s.lower():
                if i < len(sentences) * 0.33:
                    position = "前段（優）"
                elif i < len(sentences) * 0.66:
                    position = "中段（良）"
                else:
                    position = "後段（可）"
                break
    return {
        "query": query,
        "mentioned": mentioned,
        "position": position,
        "answer_preview": answer[:200]
    }

def daily_report():
    results = [check_ai_visibility(q) for q in QUERIES]
    mention_rate = sum(1 for r in results if r["mentioned"]) / len(results)

    filename = f"geo_report_{datetime.now().strftime('%Y%m%d')}.csv"
    with open(filename, 'w', newline='', encoding='utf-8-sig') as f:
        writer = csv.DictWriter(f, fieldnames=["query","mentioned","position","answer_preview"])
        writer.writeheader()
        writer.writerows(results)

    print(f"日期: {datetime.now().strftime('%Y-%m-%d')}")
    print(f"AI 引用率: {mention_rate:.1%}")
    print(f"詳細報告: {filename}")

if __name__ == "__main__":
    daily_report()
```

## 七、客戶銷售方案

### 7.1 客戶痛點分析

**三大核心痛點：**

1. **AI 隱形危機**
   - 「請打開 ChatGPT，搜尋你的公司名——如果 AI 完全不認識你，你的潛在客戶也找不到你。」
   - 在 AI 搜尋時代，不被 AI 提及 = 不存在

2. **SEO 投報率下降**
   - Google 搜尋量持續被 AI 工具分流
   - 傳統 SEO 帶來的流量每年遞減 15-25%
   - SEO 預算花了，但效果越來越差

3. **競爭對手已經開始**
   - 同業已經開始佈局 AI 能見度
   - 先行者會佔據 AI 的「記憶」，後來者更難追

### 7.2 銷售話術

**30 秒開場白：**

> 「X 總您好，我想請教一個問題——您最近有沒有用 ChatGPT 或 Google Gemini 搜尋過貴公司的產品？
> 我們發現一個現象：台灣超過 70% 的採購決策者，現在會先問 AI 再做決定。
> 但大多數台灣企業在 AI 的回答中是完全隱形的。
> 我們專門幫企業解決這個問題——讓 AI 在回答相關問題時主動推薦您。」

**現場演示法（殺手鐧）：**

1. 當場打開 ChatGPT（或客戶常用的 AI）
2. 輸入：「台灣 [客戶的行業] 推薦哪些 [客戶的產品類型]？」

### 9.1 免費工具：AI 能見度檢測器

**概念：** 建一個免費線上工具，讓企業輸入品牌名，自動檢測該品牌在各大 AI 平台的能見度。

**功能：**
1. 輸入品牌名稱
2. 自動向 ChatGPT、Gemini、Perplexity 發送 5 個標準問題
3. 分析回答中是否提及該品牌
4. 生成「AI 能見度評分」（0-100 分）
5. 提供改善建議摘要
6. 完整報告需留下 Email 才能下載

**轉化漏斗：**

```
免費檢測（獲取 Email）
    ↓
完整報告 + 改善建議（Email 自動發送）
    ↓
3 天後：案例分享 Email
    ↓
7 天後：免費 30 分鐘諮詢邀約
    ↓
諮詢中：現場深度診斷 + 方案推薦
    ↓
成交
```

## 十一、合規注意事項

### 台灣法規重點

| 法規 | 相關條文 | GEO 注意事項 |
|------|---------|-------------|
| **公平交易法** | 第 21 條（不實廣告） | 所有 AI 引用的數據必須真實可驗證，不可虛構 |
| **公平交易法** | 第 24 條（欺罔行為） | 不可操縱 AI 生成虛假的比較結果 |
| **消費者保護法** | 第 22 條（廣告真實） | 企業對廣告內容的真實性負舉證責任 |
| **個人資料保護法** | 全文 | 監測工具收集用戶資料需符合告知義務 |
| **商標法** | 第 68 條 | 在內容中提及競品時，不可構成商標侵權 |

### GEO 合規紅線

1. **不可虛構數據**：「市佔率 73%」必須有來源，否則涉及不實廣告
2. **不可批量灌水**：315 曝光的「GEO 投毒」在台灣同樣違法
3. **不可冒充第三方**：不能假裝是媒體或用戶來撰寫推薦內容
4. **不可詆毀競品**：比較內容必須客觀、有據可查
5. **不可操縱 AI**：不能利用 prompt injection 等技術手段操縱 AI 輸出
6. **維基百科利益揭露**：編輯維基百科條目必須揭露利益衝突

### 合規自查清單

- [ ] 所有引用的數據都有可靠來源
- [ ] 比較內容客觀公正，未詆毀競品
- [ ] 未使用虛假帳號或身份發布內容
- [ ] 收集用戶資料有明確告知與同意機制
- [ ] 廣告內容有明確標示（如有付費推廣）
- [ ] 維基百科編輯已揭露利益衝突
- [ ] 未使用自動化工具批量生產低品質內容
